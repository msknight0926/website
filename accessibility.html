Accessibility

When programming for a website, it is essential that you keep your webpage accessible for all those who navigate it. This serves many purposes. An accessible website allows for all users to be able to maximize their time while using it. Everything from appropriate color contrast to easy to read font can assist a user as they navigate your website.



The following is a list of programming I have done with the intention of making my website more accessible:

  • The colors on my website contrast very well. I use a light back drop with dark font. The contrast ratio ends up being 18.55:1.
  • I use an easy to read font, palantino.
  • Alt tags are on all images, videos, audio, and objects on this website.
  • I use a top navigation bar for this website so it is easier to navigate from page to page.
